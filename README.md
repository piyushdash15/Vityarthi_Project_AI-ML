# 📊 Sentiment Analysis Project

## 📌 Project Title

**Sentiment Analysis on User Comments using Machine Learning**

---

## 📖 Project Description

This project is a **Sentiment Analysis System** that classifies user comments into **Positive, Negative, or Neutral** sentiments using Machine Learning techniques.
The system is trained on a dataset of user comments and uses Natural Language Processing (NLP) for text preprocessing and classification.

This project is useful for:

* Social media comment analysis
* Product review analysis
* Feedback analysis
* Opinion mining

## ⚙️ How to Run the Project

1. Install Python (3.8 or above)
2. Install required libraries:

```
pip install -r requirements.txt
```

3. Run the Jupyter Notebook:

```
jupyter notebook sentiment_analysis.ipynb
```

## 📊 Dataset Information

The dataset contains **500 user comments** with sentiment labels:

* Positive
* Negative
* Neutral

Example:

| Comment                   | Sentiment |
| ------------------------- | --------- |
| This product is very good | Positive  |
| Worst experience ever     | Negative  |
| It is okay not bad        | Neutral   |
